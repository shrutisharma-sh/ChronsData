#  ChronosData AI

ChronosData AI is a simple web application that lets you **ask questions about your data** and **validate metadata before deployment**.

It is built to make data easier to understand and ensure basic data rules are followed.

---

##  What you can do

###  Ask your data

You can type questions like:

> Who owns the orders table?

And the app will give a clear answer like:

orders table is owned by data_team


---

###  Validate metadata

You can also check if your data is ready before deployment.

Just enter:
- table name  
- owner  

And the system will tell you:

-  if something is missing  
-  if everything is correct  

---

##  UI

The project includes a simple web interface where you can:

- Chat with your data  
- Validate metadata easily  

Open in browser: http://localhost:8080/index.html
